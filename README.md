# 《未来生存法则》

> AI 时代、全球化变局与个体竞争力重构

本书不侧重预测未来，而聚焦于**未来十年普通人真正需要建立的生存能力体系**——围绕六条主线展开：**认知升级、能力建设、财富积累、能力升级、风险管理、系统构建**。

## 全书结构

- **序言** —— 为什么这本书现在写、为什么你需要读
- **第一篇 未来已来**（第 1-3 章）——世界正在发生什么
- **第二篇 未来竞争**（第 4-7 章）——人与人的差距如何拉开
- **第三篇 财富法则**（第 8-11 章）——未来如何赚钱
- **第四篇 能力升级**（第 12-15 章）——未来最重要的能力
- **第五篇 风险应对**（第 16-19 章）——未来最大的挑战
- **第六篇 系统构建**（第 20-24 章）——构建未来竞争优势
- **附录 A-C** ——工具清单、十年路线图、能力自测

共 24 章 + 3 个附录，约 6 万字。
