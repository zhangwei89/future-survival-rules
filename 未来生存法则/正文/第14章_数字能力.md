# 第 14 章　数字能力

数字能力是 AI 时代**最容易被低估、最值钱的能力**。

过去数字能力意味着会用 Word、Excel、PPT；现在数字能力意味着**会用 AI、能看数据、能搭系统、能保护自己**。

## 一、AI 工具：从使用者到协作者

AI 工具有两个层级。

**第一层：使用** - 用 ChatGPT、Claude、文心一言等写文案、做翻译、起稿子。
**第二层：协作** - 用 Cursor、Copilot、Claude Code 做工程；用 ComfyUI、Midjourney 做设计；用 NotebookLM 做研究。

普通人的目标是：第一层熟练，第二层精通。

具体的 AI 工具地图：
- 写作：ChatGPT / Claude / Kimi / 豆包
- 图像：Midjourney / Stable Diffusion / 即梦
- 视频：Sora / Runway / 可灵 / 即梦
- 代码：Cursor / Copilot / Claude Code / Trae
- 研究：Perplexity / NotebookLM / Consensus
- 办公：Microsoft Copilot / 飞书智能伙伴

每个赛道选一个主用，搭配 1-2 个备用。

## 二、数据分析：用数字说话

数据分析的能力分三层。

**第一层：看懂数据** - 会看报表，知道关键指标
**第二层：解释数据** - 能解释数字背后的业务含义
**第三层：驱动决策** - 用数据指导行动

普通人的目标是至少做到第二层。

具体工具：
- Excel 进阶：透视表、公式、图表
- SQL 基础：能用 SQL 从数据库取数
- BI 工具：Tableau / Power BI / FineBI
- AI 辅助分析：让 AI 帮你做探索性分析

## 三、自动化：让重复的事自动发生

自动化的核心是**让机器做机器该做的事，让人做人该做的事**。

自动化的层次：
- 基础自动化：邮件模板、自动保存、批量重命名
- 流程自动化：用 Zapier、Make、n8n 串联多个工具
- 代码自动化：用 Python 脚本替代重复操作
- AI 自动化：用 AI Agent 替代多步骤任务

推荐入门路径：
- 第一阶段：用 iPhone 快捷指令、Mac 快捷指令做日常自动化
- 第二阶段：用 Zapier / Make 串联 SaaS 工具
- 第三阶段：学一点 Python，做更复杂的自动化

## 四、编程思维：像程序员一样思考

**普通人不需要会写代码，但需要会编程思维。**

编程思维包括：
- 拆解：把大问题拆成小问题
- 抽象：找到重复模式，提取共性
- 流程化：把手动过程变成自动流程
- 测试：先验证再扩展
- 版本管理：保留历史版本，可以回滚

这五个能力，用 AI 工具时同样重要。AI 是放大器，编程思维是放大器的使用方法。

## 五、数字安全：保护自己

数字时代的风险包括：
- 隐私泄露：不要在社交媒体发过多个人信息
- 账号安全：每个账号独立密码 + 二次验证
- 钓鱼识别：陌生链接、邮件先怀疑，再判断
- 数据备份：重要资料至少三份（本地 + 云端 + 离线）
- AI 幻觉：不要直接信 AI 给的事实，重要信息人工核实

数字安全不是技术问题，是**习惯问题**。养成习惯比学技术更重要。

## 章节小结

五种数字能力：
- AI 工具：从使用到协作
- 数据分析：看懂 → 解释 → 驱动决策
- 自动化：基础 → 流程 → 代码 → AI
- 编程思维：拆解、抽象、流程化、测试、版本管理
- 数字安全：隐私、账号、钓鱼、备份、AI 幻觉

> 数字能力是基础能力。最后一种基础能力是全球化能力——这是把个体杠杆放大的关键。
