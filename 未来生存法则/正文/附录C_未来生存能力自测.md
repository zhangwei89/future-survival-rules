# 附录 C　未来生存能力自测

> 这一份自测表覆盖本书六大主线对应的核心能力。**每个维度 1-10 分自评，最后计算总分与雷达图位置。**

## 使用方法

1. **诚实打分**——不要按「应该」打分，按「实际」打分
2. **基于最近 1 年的真实表现**——不是偶尔一次的高光
3. **总分 600 分**——6 大维度 × 100 分，每个维度由 4-5 个子项构成
4. **每半年自测一次**——看趋势比看绝对值重要

## 结果解读

### 0-200 分：起步阶段
- **现状**：基础能力较弱，未来 5 年大概率被边缘化
- **重点**：先补齐「学习力 + 执行力 + 抗风险能力」三项基础
- **路径**：建立基本工具栈 + 应急资金 + 运动习惯
- **时间表**：6-12 个月提升到 200-400 分

### 201-400 分：成长阶段
- **现状**：有一定基础，但能力不均衡
- **重点**：补短板——尤其是「AI 应用能力 + 全球化能力」
- **路径**：选 2-3 个最弱的维度重点突破
- **时间表**：1-2 年提升到 400-600 分

### 401-600 分：进阶阶段
- **现状**：综合能力较强，能应对大部分未来挑战
- **重点**：从「能力均衡」走向「能力突出」——打造自己的杀手锏
- **路径**：选定 1-2 个核心优势方向，建立个人品牌
- **时间表**：2-3 年提升到 600+ 分

### 601-800 分：领先阶段
- **现状**：综合能力突出，是同龄人中的少数
- **重点**：从「个人能力领先」走向「系统领先」——建立团队、杠杆他人
- **路径**：把个人能力转化为商业系统
- **时间表**：3-5 年成为某个细分领域的代表人物

## 雷达图自检

按八个维度的得分画雷达图，看看形状：

```
       学习力
        /\
       /  \
全球化 /    \ 思考力
      |    |
抗风险|    |执行力
      \    /
       \  /
        \/
       AI能力—创造力—财富能力
```

**理想形状**：八边形较为均衡，没有明显凹陷
**危险形状**：某个维度严重凹陷（< 30 分）——会成为致命短板

---

## 使用建议

1. **第一次自测**——作为起点，诚实记录
2. **每 6 个月重测一次**——看分数变化趋势
3. **重点关注最低 3 项**——这些是未来半年的提升重点
4. **不要追求总分最快上升**——均衡比单点提升重要
5. **把自测结果分享给家人或朋友**——外部监督能提高行动力

## 最后的提醒

**自测不是为了得到一个漂亮的分数，是为了看清自己的位置，然后有针对性地行动。**

分数低不丢人——重要的是从今天开始改变。分数高也不要松懈——AI 时代最大的风险是「曾经领先」的人停止进化。

每半年自测一次，每次都比上一次进步一点。10 年后回头看，你会感谢今天诚实面对自己的自己。
